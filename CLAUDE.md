## Active visual direction: Lumo Signal

When working on the storefront UI, follow the implementation brief in
`docs/lumo-signal-redesign.md`.

Visual references:

- `docs/design-references/lumo-signal-desktop.png`
- `docs/design-references/lumo-signal-mobile.png`

Treat these images as references for composition, hierarchy, color, typography,
spacing, card treatment, and responsive behavior. Do not copy game titles,
prices, or other generated placeholder content from the images. Real project
data and existing catalog/business logic remain the source of truth.

The highest-priority surfaces are the home page, header, budget selector, game
rows, and game cards. Keep the rest of the current pages visually consistent
with the same design system.
